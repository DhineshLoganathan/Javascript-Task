This is a simple Task Management Application built using HTML, CSS, and JavaScript. It allows users to add, edit, delete, and view tasks. Tasks can be marked as completed or pending, and users can filter tasks based on their status. All tasks are saved in the browser's local storage, so they persist even after refreshing the page.

Features
Add Task: Add a task with a name and description.

View Tasks: See all tasks in a list with their details.

Edit Task: Update the name and description of a task.

Delete Task: Remove a task from the list.

Task Status: Mark tasks as completed or pending.

Filter Tasks: Filter tasks by status (All, Pending, Completed).

Local Storage: Tasks are saved in the browser, so they don't disappear when you refresh the page.

How to Use
Add a Task:

Enter a task name and description in the input fields.

Click the "Add Task" button.

Edit a Task:

Click the "Edit" button on the task you want to update.

Enter the new name and description in the prompt.

Delete a Task:

Click the "Delete" button on the task you want to remove.

Mark as Completed/Pending:

Click the "Complete" button to mark a task as completed.

Click "Undo" to mark it as pending again.

Filter Tasks:

Use the filter buttons at the top to view:

All tasks.

Only pending tasks.

Only completed tasks.

How to Run
Download or clone the project files.

Open the index.html file in your browser.

Start managing your tasks!

Technologies Used
HTML: For the structure of the application.

CSS: For styling and making the app look modern and responsive.

JavaScript: For adding functionality like adding, editing, deleting, and filtering tasks.
